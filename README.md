# 所属研究室のホームページをリニューアルさせていただきました

研究室ホームページについて、

- 研究内容が一目で伝わりにくい
- スマートフォンで見ると文字が小さい

という課題を感じたため、リニューアルを提案しました。<br>
サイトの閲覧者だけでなく、実際に更新作業をする先生にも使いやすいサイトを作ることを意識しました。

大学のサイトではHTMLとCSSのみしか使えないようなので、その二つの言語のみ使用しています。

## 工夫した点

- トップページで研究内容を把握しやすいように画像を背景に挿入し、VRが映っているものにした
- スマートフォン・iPad表示への対応（レスポンシブ対応）
- 更新頻度が高いnews、研究業績、memberの3セクションはiframeで別htmlファイルにした
